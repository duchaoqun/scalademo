## Chapter 15 注解

### 【Demo01】什么是注解？

- 注解可以让你在程序的各个条目中添加信息，这些信息可以被编译器或者外部工具处理。
- 注解是那些你插入到代码中以便有工具可以对他们进行处理的标签。
- 工具可以在代码级别运作，也可以处理被编译器加入了注解信息的类文件。

### 【Demo02】什么可以被注解？

- 在 Scala 中，你可以为类、方法、字段、局部变量、参数添加注解，跟 Java 一样。
