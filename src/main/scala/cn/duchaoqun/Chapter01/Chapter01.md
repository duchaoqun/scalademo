### 【Demo08】基本类型 Option

- 标准类库中的 Option 类型用样例类来表示那种可能存在、也可能不存在的值。
- Option 类型的值通常作为 Scala 集合类型(List,Map等)操作的返回类型。
- Option 的子类型 Some
- Option 的子类型 None
- Option 通常会与 match case 表达式一起使用。
